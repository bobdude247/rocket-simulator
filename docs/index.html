Rocket Simulator

Mission simulator with two modes: an SLS-inspired heavy vehicle and a hobby rocket profile.

Run Locally

python -m src.rocket_simulator.cli --vehicle sls --destination mars --time-compression 86400 --max-events 20

Modes

  • sls: heavy launch + capsule + long missions
  • hobby: small suborbital rocket with parachute recovery

Destinations

earth, moon, mercury, venus, mars, jupiter, saturn, uranus, neptune

Source code: github.com/bobdude247/rocket-simulator
